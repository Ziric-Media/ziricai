# Product Updates
> Category: product-updates | ZiricAI Platform Knowledge Base

## Q: Where are product updates announced?
**A:** Portal release notes, email newsletter, and knowledge category 25.

## Q: How often is ZiricAI updated?
**A:** Biweekly improvements plus major quarterly feature releases.

## Q: Do updates require downtime?
**A:** Most deploys zero-downtime on Firebase hosting.

## Q: How do I see changelog?
**A:** Settings → What's New in portal lists recent changes.

## Q: Are breaking API changes notified?
**A:** 30-day notice on deprecated API versions via email and docs.

## Q: Can I beta test features?
**A:** Opt into beta programme in portal Settings → Labs.

## Q: Recent major features?
**A:** Expanded Marketplace packs, Sarah improvements, analytics dashboards, Paystack integration.

## Q: How do I request roadmap priority?
**A:** Vote on feedback board and discuss with account manager on Business+.

## Q: Are updates included in subscription?
**A:** All platform updates included — no extra fee for standard features.

## Q: Do industry packs update separately?
**A:** Pack template updates deploy automatically; review release notes.

## Q: Mobile portal improvements?
**A:** Responsive portal updates ship continuously — native app on long-term roadmap.

## Q: Security patch policy?
**A:** Critical patches within 24 hours; routine updates weekly.

## Q: Where is API version history?
**A:** Documentation → API → Changelog.

## Q: Do updates affect my automations?
**A:** Breaking workflow changes rare — preview in beta before production.

## Q: How to stay informed?
**A:** Enable release notification emails in profile settings.
